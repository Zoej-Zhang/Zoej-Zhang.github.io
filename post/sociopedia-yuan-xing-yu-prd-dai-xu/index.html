Sociopedia-原型与 PRD （待续

2021-03-08

背景
Sociopedia 是一个社会事务信息聚合平台，为缺少社会经验的年轻人提供了解问题、解决问题的平台。搜索是其最主要的功能。
详细介绍见 brief。

主要流程

页面展示交互逻辑

  1. 首页
  • 首页信息结构
  • 首页原型图
  • 展示元素
    • 「我的」icon
      • 点击进入个人设置页面
    • 产品 logo
    • 当前城市
      • 基于用户 GPS 位置显示，点击可更换城市
    • 搜索框
      • 默认显示「请输入」
      • 点击搜索按钮或键盘回车键进入搜索结果页
    • 分类模块
      • 点击任意分类弹出子类面板

1.1 个人设置页面

  • 触发条件
    • 点击首页「我的」icon
  • 展示元素
    • 用户头像及 ID
    • 消息中心
      • 点击进入消息中心页面
    • 我的收藏
      • 点击进入我的收藏页面
    • 设置
      • 点击进入设置页面

1.2 内容分类面板

  • 前置条件
    • 点击首页内容分类模块
  • 收起条件
    • 点击面板外任意位置，收起面板回到首页
  • 点击面板内子主题
    • 点击进入该标题相应的搜索结果页面
  1. 搜索结果页
  • 返回
    • 点击返回首页
  • 搜索框
    • 点击可输入文字重新搜索
  • 城市选择
    • 点击切换城市，切换后根据所选目的地刷新搜索结果
  • 结果分类
    • 点击选择结果分类
  • 排序
    • 点击可按照相应规则对搜索结果进行排序
  • 筛选
    • 点击可按照相应规则对搜索结果筛选
  • 结果列
    • 显示内容标题和更新时间，点击进入详情页
  • 反馈
    • 位于所有搜索结果后，点击弹出反馈弹窗
  1. 内容详情页
  • 返回
    • 点击返回上一页面
  • 文章模块
  • 文章标题
  • 分享
    • 点击弹出分享面板
  • 收藏
    • 点击将文章收藏至「我的收藏」
  • 文章更新日期
  • 文章内容
    • 文章内容以小标题为界，标题位于该段内容前，显示编辑 icon，点击该 icon 可提交内容修改，审核通过后更新内容显示在该文章中
  • 目录
    • 目录为悬浮按钮，不跟随页面滑动。点击展示文章目录，即各级标题。点击目录内标题直接跳转至该部分
